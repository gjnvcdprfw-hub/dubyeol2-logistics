## 2. 로그인 대시보드 화면별 구조

### 2.1 마이페이지 홈 (`/dashboard`)
위에서 아래로 카드 섹션:
1. **예치금 현황** — 사용 가능 잔액 ₩ + [관리][충전하기][내역보기]
2. **구매대행 진행 현황** (n건, 전체보기) — 상태 칩 5개: 견적중 / 견적완료 / 구매중 / 발송대기 / 발송완료
3. **입고 관리 현황** (n건, 전체보기) — 입고대기 / 입고완료 / 출고완료 / 에러 제품 / 반송대기
4. **출고 관리 현황** (n건, 전체보기) — 포장중 / 결제대기 / 출고대기 / 출고완료
5. **반품 관리 현황** (n건, 전체보기) — 처리대기 / 승인 / 거절 / 완료
6. **Smart Order 카드** — 등록 상품 / 확인 필요 / 주문 준비 / 주문 진행중 + [상품 보드 열기]

### 2.2 구매대행 관리 (`/dashboard/purchase-management`)
상태 탭: 결제대기 / 견적중 / 견적완료 / 구매중 / 셀러발송대기 / 셀러발송완료 / 환불완료
(가이드 문서의 상태 흐름: 견적중 → 견적완료 → 구매진행 → 발송대기 → 발송완료 → 창고입고)

상태 탭: 입고대기 / 입고완료 / 발송완료 / 에러제품 / 반송대기

### 2.6 출고 관리 (`/dashboard/shipment`)
상태 탭: 포장중 / 발송대기 / 결제대기 / 발송예정 / 발송완료

### 2.7 반품 관리 (`/dashboard/warehouse-management/returns`)
상태 탭: 전체 / 처리대기 / 승인 / 거절 / 처리중 / 완료. 안내: 반품 요청 → 창고 담당자 승인/거절 → 승인 시 환불액 예치금 자동 충전, 영업일 2~3일 소요.

### 2.8 창고 주소 (`/dashboard/shipping-address`)
- 회원별 고유 **입고ID** 발급(8자리 영숫자), 주소 끝에 자동 부착 → 입고 시 상품 구분 키
- 중국 창고 주소(웨이하이, 수령인·연락처·주소) + [원클릭 복사] [개별 복사]
- 사용법 3단계 + 주의 문구

### 2.9 출장검품 주문 조회 (`/dashboard/inspection`)
검품감사 신청 현황 리스트. 상세 화면 구성(서비스 소개 페이지의 미리보기 기준): 예약번호(DL-YYYYMMDD-INSnnn), 상태 스테퍼 5단계(신청접수 → 일정조율 → 견적및결제 → 검품진행 → 검품완료), 탭(기본정보/공장정보/일정관리/보고서), 특별 요구사항 한중 병기(자동 번역).

### 2.10 내 프로필 (`/dashboard/profile`)
탭 4개: 기본 프로필(회사명·회사명 중문·담당자명·연락처·이메일·비밀번호 변경) / 배송 정보 / 세금계산서 자료 / 알림 설정

### 2.11 예치금 관리 (`/dashboard/wallet`)
잔액 카드(₩, "즉시 사용 가능") + [충전하기][환불 신청] + 통계 3개(총 충전/총 사용/거래 횟수) + 탭(거래 내역/충전 대기)

### 2.12 장바구니 (`/dashboard/cart`)
빈 상태: "장바구니가 비어있습니다" + [쇼핑 계속하기][구매 관리]

## 3. 요율 정본 (두리무역 초기값 — 대표님 추후 컷팅)

### 3.1 수수료
- 구매대행 수수료: **상품가 5% + 수수료의 VAT 10%** (예: 10만원 → 5,000 + 500)
- 배송대행 수수료: **0%** (운임에서 마진)
- 결제: 원화 계좌이체 또는 예치금 (카드 미지원, 개발 중 표기)

### 3.2 국제 배송비
| 방식 | 요율 | 리드타임 |
|---|---|---|
| 해운 택배 | 첫 1kg ¥25 + 추가 kg당 ¥5 | 약 5~7일 |
| 항공 택배 | 서류비 ¥32.5 고정 + 100g당 ¥1.8 (0kg부터) | 약 3일 |
| LCL 해운(사업자) | $65~70/CBM, 최소 0.5CBM + 중국수출신고 ¥180 + DO ₩20,000 + 통관수수료 ₩30,000(고객 관세사 지정 시 면제) | 약 3~5일 |
- 부피중량 계수 6000, 실중량 vs 부피중량 중 큰 값 과금
- 간이통관비 ₩3,000 (USD 150 이하)
- 대한통운 추가비: 2-5kg/80-100cm ₩600 · 5.1-10kg/101-120cm ₩1,200 · 10.1-15kg/121-140cm ₩1,800 · 15.1-20kg/141-160cm ₩2,400
- 소포 20kg+ 경동택배 착불 / LCL 국내운송 용달·퀵 착불
- ¥ 단가는 결제일 당일 환율 적용

### 3.3 창고 보관료
30일 무료 → 31~60일 ₩500/일/박스 → 61일+ ₩1,000/일/박스 → 90일+ 폐기 가능

### 3.4 검수/검품 (우리 확정 정책의 근거)
- **기본 무료(구매·배송대행 포함)**: 입고 사진 1~2장 + 외포장 이상 안내. "전수·샘플링·AQL·기능·치수·개별검사 아님" 명시.
- **창고 검수(유료 부가)**: 검수 ¥35/시간(사진 2장 포함) 또는 ¥1/개 표기 병존(페이지 간 차이), 추가 사진 ¥1/장, 동영상 ¥5/분
- **출장검품(전문 검품, 별도 서비스)**: 1~4일 29만원/일 · 5~9일 27만원/일 · 10일+ 25만원/일, 보고서 24시간 내. 유형: 품질검품/공장감사/선적검품.

### 3.5 부가서비스 단가 (개당, ¥)
- 원산지: 스티커 0.3 / 라벨(봉제) 0.6 / 도장 0.6 / 행택 0.5
- 판매표기: 바코드 0.5 / KC 0.5 / 식품검역 0.5 / 14세표기 0.5 / 커스텀행택 0.5 / 세탁라벨 0.6
- 포장: OPP소 0.3 / OPP대 0.5 / 에어캡 2 / 묶음포장 1 / 묶음옵션 0.3 / 태그제거 0.2 / 포장제거 0.3
- (서비스 소개 페이지 원화 표기 병존: 원산지 스티커 60원/개, 라벨 미싱 120원/개, OPP 60원/개)

### 3.6 재포장 부자재 (원화)
- 박스(포장비 포함): 미니 ₩1,157 / 소 ₩2,314 / 중 ₩3,471 / 대 ₩4,627
- 포장재: PE봉투 ₩224 / 에어캡 ₩463/m / OPP중 ₩116 / OPP대 ₩186 / OPP소 ₩67 / 테이프보수 ₩1,189 / 맞춤라벨 ₩119 / 충전물 ₩476 / 신축필름 ₩2,378
- 라벨: 부착 ₩68 / 봉합 ₩135
- 기타: 팔레트 ₩40,135 / 박스 소분 ₩476 / 포대 소분 ₩238
- "자동 포장" 옵션: 창고가 최적 포장재 선택, 패킹 후 비용 확정

### 3.7 견적서 구성 (구매대행 1차 결제)
- **포함**: 상품가(확정단가×수량, 당일 환율 환산) + 중국 내 배송비 + 대행 수수료 + 수수료 VAT 10% + (선택)부가서비스 기본 비용
- **미포함(별도)**: 국제운임(출고 시 확정) / 관세·부가세(통관 시) / 부가서비스 추가 / 환율차 / 보관료(30일+)
- 상세 탭: 기본정보 / 가격정보(확정단가·적용환율·상품총액·중국배송비·수수료·수수료부가세·1차결제금액) / 배송정보(1688주문번호·구매일·운송장) / 통관정보(HS코드·관세율·주의사항)

## 4. 기능 인벤토리 (메뉴 밖 기능)

### 4.1 1688 검색·구매대행
- 한국어 키워드 검색(AI 중국어 번역) + 이미지 검색 + 카테고리 브라우즈
- 상품 카드: ¥가격 + ₩환산 + 월 판매량 + 재구매율 + 뱃지(드롭쉬핑/우수/파워셀러)
- AI 이미지 번역(상세페이지·사이즈표), 상세페이지 전체 번역
- 장바구니 → 일괄 견적 요청(부가서비스 함께 선택) → 직원 견적 → 예치금 결제
- 내 상품 리스트: 구매 이력 자동 저장, 원클릭 재구매, 엑셀 업로드, 일괄 견적
- 회원가입 없이 검색 가능

### 4.2 배송대행 자동화
- 1688 OAuth 계정 연동 → 주문 목록 자동 동기화(주문번호/SKU/이미지)
- 중국 운송장 실시간 자동 조회·입고 예정일 자동 계산
- 타오바오 모바일 공유 URL 자동 인식(상품명·가격·옵션 자동 파싱)
- AI HS코드 추천(기본관세·WTO·한중FTA 비교, 최종 판단은 관세사·세관 우선 명시)
- 자체 WMS: 바코드 부여·스캔, 입고 사진, 재고 위치, 실시간 대시보드 (별도 도메인 wms.duly.co.kr)
- 3PL 풀필먼트(재고 보관 + 개별 배송)

### 4.3 스마트오더 (Excel 대량 구매 자동화)
- 플랫폼(스마트스토어/쿠팡) 주문 Excel 업로드 → AI 파싱(1688 링크 포함 행 구조화) → 1688 옵션 매칭 추천 → 고객 확인 → 직접구매(수수료 0, 고객 1688 계정) 또는 구매대행(5%+VAT) → 입고·출고 추적 → 국내 운송장 회신 Excel 생성
- 매칭 라이브러리(이전 매칭 이력 재사용)

## 5. UI/외관 벤치마크 노트 (외관 그대로 벤치마킹용)

- **랜딩 구성 순서**: 히어로(가치제안+검색창+신뢰 뱃지 3개) → 1688 검색·구매대행 섹션 → 배송대행 5단계 플로우 → 스마트오더 4단계 → 검품감사(도시 목록+리포트) → 실적 숫자(수수료 비교·연차·월 처리·고객사) → FAQ 아코디언 → CTA(무료 가입) → 푸터(서비스/고객지원/회사정보 3열 + 사업자 정보)
- **대시보드 레이아웃**: 좌측 고정 사이드바(그룹 헤더 이모지 + 메뉴) + 상단 수평 공개 메뉴 + 본문 카드 그리드. 현황 카드는 "제목 + n건 + 전체보기 + 상태 칩 나열" 패턴 통일.
- **상태 칩 패턴**: 모든 관리 화면이 상태 탭/칩 필터 + 빈 상태 문구("해당 상태의 상품이 없습니다") 패턴
- **신뢰 지표 반복**: "8년+ / 50,000+ 누적 주문 / 1,000+ 고객사 / 실시간 상담" 4종 세트가 거의 모든 페이지 하단에 반복
- **가격 투명성 표현**: 요율표를 공개 페이지에 노출 + "참고용, 최종 금액은 확정 후" 면책 문구 병기
- **로그인**: Google/카카오 간편 로그인 + 이메일, 고객/관리자 로그인 분리(`/auth/customer/*`)
- **책임 경계 명시 패턴**: 적합한 고객/적합하지 않은 경우, 비용 경계, 통관·책임 경계를 공개 페이지에 명시 (분쟁 예방 카피)
